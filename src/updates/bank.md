### Components

**Ghost surfaces** — Any section can go dark. Add `.ghost` to flip the color scale — text, borders, and backgrounds all invert without touching a single variable. [Component]

**Cycling animation** — Timed rotation across a set of images or divs. Configurable speed, hold duration, and frame count. Used for multi-brand splashes and product reveals. [Component]

**Video hero** — Full-bleed autoplay video with a ghost overlay for text and CTAs. Drops gracefully to a static background when video fails. [Component]

**Chip** — Small status pill with a colored dot. Signals things like "accepting early access" or category without taking up space. [Component]

**Step indicators** — Numbered steps for how-it-works sequences. Stay centered, stay circular, never stretch. [Component]

**Browser mockup** — Simplified browser chrome for showing product screenshots or live previews without a real browser frame. [Component]

**Audio player** — Play, pause, and scrub. Minimal controls that fit the minterface type scale and respond to dark mode. [Component]

**Markdown renderer** — Drop in a string of markdown, get back formatted HTML. Used for forum posts, release notes, and any user-generated content. [Component]

**Dynamic project grid** — JSON-driven card grid. Point it at a data file, get back a rendered list with no markup changes. [Component]

**Theme toggle** — One button. Reads system preference on load, lets the user override it. Sun and moon swap on click. [Component]

**Voice input** — Microphone button that transcribes speech in real time. Feeds directly into any text field. Built for Quictate, reusable anywhere. [Component]

**Streaming text** — Renders tokens as they arrive from an LLM. Cursor blinks at the insertion point, scrolls to follow output. [Component]

**Prompt composer** — Textarea with a send button and optional file attachment. Handles submit on Enter, newline on Shift+Enter. [Component]

**Tool call card** — Collapsible card that shows what an AI agent called, with what arguments, and what it returned. Good for transparency in agentic UIs. [Component]

**Confidence indicator** — A subtle visual signal — color, opacity, or a small badge — that communicates how certain a model is about its output. [Component]
